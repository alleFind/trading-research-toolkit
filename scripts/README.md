# scripts/

每个 Sprint 完成时往这里加对应脚本。

## 目录

| 目录 | 内容 | 加入 Sprint |
|------|------|-----------|
| `zsxq/` | 知识星球增量同步 + 软链入库 | S3 |
| `wechat/` | 微信群清洗 + 每日提取 | S4 |
| `common/` | 多源通用工具（如统一软链脚本） | S3+ |

## 约定

- 所有脚本必须从环境变量或 `.local.*` 文件读敏感信息，不写死
- 所有脚本必须可独立执行（`./script.sh --help` 给出说明）
- 跨平台优先（macOS + Linux），如果只支持 macOS 在脚本头部注明
- 必须 idempotent（重跑不会出错）
- 必须支持 `--dry-run`
